# Fundamentos de Programación · UNICARIBE

Presentación interactiva sobre fundamentos de programación. Un único archivo `index.html` con todo embebido (HTML + CSS + JS), pensado para desplegarse como sitio estático.

## Demo

Despliegue en Cloudflare Pages.

## Contenido

15 diapositivas que cubren:

- Qué es la programación y los algoritmos
- Lenguajes de programación (Python, Java, C++, JavaScript)
- Lenguajes de bajo vs alto nivel
- Compilados vs interpretados
- Tipos de datos
- Lógica binaria (con conversor decimal→binario interactivo)
- Variables y constantes (con demos interactivos)

## Navegación

- **Flechas ←/→** o **Espacio** — siguiente / anterior diapositiva
- **Home / End** — primera / última diapositiva
- **Click** en los puntos inferiores — saltar a diapositiva específica
- **Swipe** en móvil
- **`?slide=N`** o **`#N`** en la URL — deep-link a una diapositiva

## Desarrollo local

No requiere build. Abrir `index.html` en cualquier navegador moderno:

```bash
xdg-open index.html
```

O servir con cualquier servidor estático:

```bash
python3 -m http.server 8000
# Luego visitar http://localhost:8000
```
